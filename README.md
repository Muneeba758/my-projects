# 🎯 Quiz Website

## 📌 Project Description

This is a simple quiz website developed using HTML, CSS, and JavaScript.
It allows users to attempt quiz questions and get their results instantly.

## 🚀 Features

* Multiple choice questions
* Instant result display
* Simple and user-friendly interface
* Interactive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Files

* index.html → Main structure of website
* style.css → Design and styling
* index.js → Quiz functionality

## ▶️ How to Run

1. Download or clone the repository
2. Open **index.html** in any web browser
3. Start the quiz
